6) UI / UX SYSTEM (APPLE-LEVEL DESIGN)
Design philosophy
“Zero cognitive load”
“One action per screen”
“Trust-first visuals”
Design system

Typography

Primary: Inter (system clean)
Secondary: Noto Sans Ethiopic (Amharic support)
Optional: SF Pro (iOS feel)

Colors

Success: Green (job completed)
Warning: Amber
Danger: Red (SOS / dispute)
Neutral: Slate/White

Components

Job cards (swipeable)
Trust score badge
Map-based job tracking
Floating SOS button
In-chat action buttons

Motion

micro transitions (150–250ms)
swipe gestures for accept/reject
